Blog-Redesign

The task to redesign the more or less unstyled wordpress blog was the first exercise of the webtypography course.

The Design – What’s New?

The blog’s structure should remain the same. Multiple colors act as an important design element to mark the different articles. Every blog-post should get a preview image, even if the author does not provide one. The image should then be automatically generated. How this is done is dependent of the article-type. A tweet-post should just show the twitter-name of the tweet’s author. A code fragment should use a fraction of the actual code and more vague article types should just show the first letter of the paragraph. This should provide more visual anchors which could become important as soon as the blog is widely populated with blog-posts.

Further two different typefaces should achieve some hierarchy. The serif typeface “Vollkorn” developed at FH Potsdam is used in headlines and the relatively calm “Lato” is used for text.

Quick Look

Color palette

Each article type should get its own color for better recognition. The colors should blend well with each other but also be easily distiguishable.
