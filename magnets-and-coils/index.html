A Marble Run Game

Jonas Scheiwiller
ZHdK

During the school project in interaction design at the ZHdK, the topic 'energy harvesting' was centered. We approached this topic in a playful way and built a marble run game, where two players can compete against each other. Short reaction time and good anticipation wins.
